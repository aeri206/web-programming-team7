# 💸 주린이를 위한 안내서

## 🌎 시작하기
`rails server`

## 🧩 프로젝트 개요
진짜 주식 초보, 주린이를 위한 백과사전! <br />
주식 관련 용어부터 주식 대가들의 투자까지 한눈에 살펴보고, 직접 조건을 선택하여 알맞은 기업을 스크리닝해볼 수 있습니다.
